# Dawn Forest - Desenvolvido na Godot Game Engine | Versão: 3.4.4

Jogo de Plataforma 2D em Desenvolvimento para um Curso da Udemy.

## Principais Mecânicas (Personagem):

- Tomar Dano | Morrer.
- Movimentação Horizontal.
- Atacar, Defender, Agachar.
- Animações para todos os Estados.
- Movimentação Vertical (Pulo, Pulo Duplo, Pulo na Parede).

## Principais Mecânicas (Inimigos):
- Tomar Dano | Morrer.
- Animações para todos os Estados.
- Atacar o Personagem (Quando no Alcance)
- Perseguir o Personagem (Quando no Alcance)

## Sistemas:
- Sistema de Diálogo.
- Sistema de Status | Exp.
- Sistema de Drop de Itens.
- Barra de Vida | Mana | Exp.
- Sistema de Spawn de Monstros.
- Sistema de Inventário com 27 slots.
- Sistema de Compra e Venda de Itens (Npc Mercador).

## Sistema de Serialização:
  - Status.
  - Inventário.
  - Itens Equipados.
  - Ouro do Jogador.
  - Vida | Mana | Exp Atuais.
  - Último Checkpoint salvo pelo Jogador (Posição).
